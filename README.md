# Music Score Viewer

A Python application to view, navigate, and annotate PDF music scores.

## Features
- Zoom-to-fit display
- Side-by-side page view
- Annotations (Pen, Text, Eraser)
- Metadata search (Composer, Title, Tags)

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python music_score_viewer.py`
